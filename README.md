# passGenerator written in golang

*mainly for self development purpose.*

__Usage:__ 
  * -d int
        Number of password digits. (default 8)
  * -s    Include symbols.
  * -u    Include upper case digits.</p>

__Example output:__
$ ./passGen -d 20 -u -s  

Digits: 20 \
Symbols: true \
Upper case: true  

Password: !N}o65f|r06Ep'@/w$LB
