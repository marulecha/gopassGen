# passGenerator written in golang

*italic* mainly for self development purpose.

Usage: 
$ ./passGen -d 12 -s -u
  -d int
        Number of password digits. (default 8)
  -s    Include symbols.
  -u    Include upper case digits.</p>

Example output:

$ ./passGen -d 20 -u -s

Digits: 20
Symbols: true
Upper case: true

Password: !N}o65f|r06Ep'@/w$LB
